Admin Rights

msiexec silence

Sometimes you want to bypass have the correct rights. App in 2016 this still works on Windows 10. When trying to install nodejs and python via downloading the msi and clicking on this msi, I was interrupted from installing. Just run via the CLI in silent mode to bypass this. C:\development\apps>msiexec /a python-x.x.x.msi /qb TARGETDIR=C:\development\apps\python Thanks Sylvian
